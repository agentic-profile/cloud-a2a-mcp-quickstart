Cloud Ready A2A and MCP Quickstart

This server provides example Agent-to-Agent (A2A) and Model Context Protocol (MCP) services that support Universal Authentication and can be easily deployed to AWS for massive scalability, reliability, and availability.

Universal Authentication is a lightweight combination of Decentralized Identifiers (DID) and JSON Web Tokens (JWT), riding over HTTPS. It enables discovery, authentication, and secure communication between any two agentic systems on the Internet and does not require centralized Authentication Services.

  • This server is open source and available on GitHub
  • Demo available at demo.universalauth.org
  • More informmation available from Agentic Profile blog
